# CodeBotJS

## A Discord Bot allowing Users to execute Sandboxed JavaScript Code

### Install dependencies:
```
npm i
```

### Create `token.js`:
```js
const token = "YOUR_BOT_TOKEN_HERE";
exports.token = token;
```

### Create `functions.db`:
```
touch functions.db
```

### Run the bot script:
```
node bot.js
```
